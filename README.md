# Gemini API を使用した RSS ニュース表示Webサイト

このWebサイトは、RSSから最新ニュースを取得し、
カテゴリ選択に応じてニュース一覧を動的に表示するアプリ。
各ニュースには Gemini API を用いたタグ生成処理を実装している。

---

## カテゴリ変更から画面表示に至るまでのロジック

1. ユーザーが画面上のプルダウンからカテゴリ（ゲーム・スポーツ・音楽）を選択する
2. 「ニュースを取得」ボタンを押すと、JavaScript が選択されたカテゴリを取得する
3. フロントエンドから FastAPI に対して HTTP リクエストを送信する
    （例：`/news?category=game`）
4. バックエンドでは、選択されたカテゴリに対応する RSS フィードを取得する
5. RSS フィードから最新ニュースを10件以上抽出する
6. 抽出したニュース1件ごとに Gemini API を呼び出し、ニュース内容に基づいたタグ生成を行う
7. ニュース情報と生成したタグを JSON 形式でフロントエンドへ返却する
8. フロントエンドは受け取ったデータをもとに、ニュース一覧を動的に画面へ表示する

---

## Webサイトを動作させるための起動手順

### 1. 必要な環境

- Python 3.10 以上
- Google Gemini API の APIキー
- VSCode (推奨)

---

### 2. 事前準備 (インストール)

以下のコマンドを使用して、必要なライブラリをインストールする。

```bash
cd backend
pip install -r requirements.txt
```

---

### 2-1. 環境変数の設定
backend フォルダに .env ファイルを作成し、
以下の形式で Gemini API の APIキーを記述する。

```GEMINI_API_KEY=自分のAPIキー```

※APIキーはセキュリティの観点からGitHubには含めない。

---

### 2-2. バックエンドの起動
以下のコマンドで FastAPI を起動する。

```uvicorn main:app --reload```

起動後、以下のURLで APIが動作していることを確認できる。

```http://127.0.0.1:8000/categories```

---

### 2-3. フロントエンドの起動
frontend フォルダ内の index.html を
VSCode の Live Server (Go Live) を使用して起動する。

ブラウザ上でカテゴリ選択とニュース取得が可能となる。
